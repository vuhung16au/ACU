# Key Takeaways

- MongoDB stores JSON-like documents instead of relational rows.
- `MongoDB.Driver` provides strongly typed async CRUD operations in C#.
- Docker Compose gives a repeatable local MongoDB environment.
- ObjectId values are commonly represented as strings in Razor Pages models.
- Repository pattern keeps Mongo-specific logic out of PageModels.
- User Secrets and environment variables should be used for connection strings.
- This project helps compare NoSQL document workflows with EF Core relational workflows.
