# Hello World CLI - C# Console Application

A simple command-line C# application that demonstrates the basic structure of a .NET console program. This project is designed for students learning C# and .NET development for the first time.

## Overview

This is a minimal "Hello World" console application that introduces the fundamental concepts of C# programming:
- Program structure and entry points
- Namespaces and classes
- The `Main` method
- Console output using `Console.WriteLine()`

## Learning Objectives

By exploring this project, you will learn:

1. **Project Structure**: Understanding the basic files needed for a .NET console application
2. **C# Syntax**: Learning the fundamental syntax of C# including namespaces, classes, and methods
3. **Entry Point**: Understanding how the `Main` method serves as the application's entry point
4. **Console I/O**: Using the `Console` class to write output to the terminal
5. **Build Process**: Compiling and running C# code using the .NET CLI tools

## Prerequisites

Before you begin, ensure you have:

- **.NET 10.0 SDK** installed on your system
- A **code editor** such as Visual Studio Code, Visual Studio, or Rider
- Basic understanding of **command-line interfaces** (Terminal/Command Prompt)

## Technology Stack

- **Language**: C# 
- **Framework**: .NET 10.0 (verified)
- **Project Type**: Console Application
- **Development Environment**: macOS, VS Code

## What This Program Does

When you run this program, it performs one simple action:
1. Outputs the text "Hello World!" to the console
2. Exits successfully

While simple, this program demonstrates all the essential components of a C# console application.

## Next Steps

After completing this project, you can:

2. **Add user input**: Use `Console.ReadLine()` to get input from the user
3. **Add logic**: Implement conditions, loops, and more complex functionality
4. **Create methods**: Break down functionality into separate methods
