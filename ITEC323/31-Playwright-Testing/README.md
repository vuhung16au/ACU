# 31-Playwright-Testing

## Overview

This module introduces browser automation with **Playwright** in a simple, beginner-friendly way.

Students learn how to:

- automate a Razor Pages form submission
- capture screenshots during a test
- record browser actions as a `.webm` video
- optionally convert the recorded video into a `.gif`

## Why This Module Matters

Playwright helps students see that testing is not only about checking backend logic. It can also:

- open a real browser
- click buttons and type text like a user
- confirm what appears on screen
- capture evidence of a successful test run

## Project Structure

- `01.RazorPagesPlaywrightDemo` - a simple Razor Pages form and matching Playwright automation tests

- .NET 10
