# Playwright Test Project

This test project automates the sample Razor Pages app with Playwright.

## What It Covers

- open the home page
- fill in the form
- submit the form
- verify the submitted values
- save a screenshot
- save a `.webm` recording
- optionally create a `.gif`

## Output Folders

- `artifacts/screenshots`
- `artifacts/videos`
- `artifacts/gifs`

## Notes

Playwright records video as `.webm`.

The `.gif` output is optional and depends on `ffmpeg` being installed on the computer.
