# Kotlin Hello World

A simple Hello World application written in Kotlin to demonstrate basic project setup and execution.

## Overview

This project serves as a minimal starting point for Kotlin development. It demonstrates:
- Basic Kotlin syntax with a simple `main()` function
- Gradle build configuration using Kotlin DSL
- Standard project structure for Kotlin applications

## Project Structure

```
kotlin-hello-world/
├── src/
│   └── main/
│       └── kotlin/
│           └── HelloWorld.kt
├── build.gradle.kts
├── settings.gradle.kts
└── README.md
```

## Requirements

- Java Development Kit (JDK) 11 or higher
- Gradle 7.0+ (or use the Gradle wrapper)
