# 33-SPA-Integration

## Overview

This module introduces Single Page Application integration by pairing an Angular frontend with an ASP.NET Core backend.

The module uses `Microsoft.JavaScript.Templates` with the `angularwebapi` template because the installed `.NET 10` SDK does not provide `dotnet new angular`.

## Learning Objectives

By working through this module, students will learn how to:

- scaffold an Angular and ASP.NET Core sample with `angularwebapi`
- understand local development flow versus published deployment flow
- connect Angular components and services to ASP.NET Core API endpoints
- inspect JSON requests and responses between frontend and backend
