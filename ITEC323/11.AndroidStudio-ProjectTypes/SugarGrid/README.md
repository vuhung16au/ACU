# SugarGrid

SugarGrid is a modern, vibrant dessert recipe application designed to provide a seamless browsing experience across various Android form factors. This project showcases a responsive grid layout, efficient data handling with MVVM, and the latest Material 3 design principles.

> **Note**: This project was implemented with the assistance of the **Android Studio (Gemini) AI** feature, demonstrating AI-powered collaborative Android development.

## 🍰 Project Overview

SugarGrid focuses on visual appeal and ease of use. It features a dynamic home screen with a responsive grid of dessert recipe cards, high-quality imagery, and an intuitive filtering system. The app is optimized for both phones and tablets, ensuring a professional look on any screen size through its adaptive layout.

## 🎓 Learning Objectives

This project serves as a practical implementation of several core Android development concepts:
- **Jetpack Compose**: Building modern, declarative UIs with components like `LazyVerticalGrid` and `Scaffold`.
- **MVVM Architecture**: Separating concerns using ViewModels and StateFlow for robust state management.
- **Material 3 Design**: Implementing dynamic color schemes, expressive typography, and full edge-to-edge displays.
- **Image Loading**: Efficiently fetching and displaying remote images using the Coil library.
- **Responsive Design**: Creating layouts that adapt gracefully to different screen dimensions and orientations.

## 🛠 Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material Design 3)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Concurrency**: Kotlin Coroutines & Flow
- **Image Loading**: Coil
- **Dependency Injection/Management**: Gradle Version Catalog (libs.versions.toml)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Android Studio**: Ladybug (2024.2.1) or newer.
- **JDK**: Java Development Kit 17.
- **Android SDK**: API Level 36 (Android 15) or newer for the best experience.
