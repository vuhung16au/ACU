# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 01.BasicFetchAPI
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the learning goals and links to the JSON endpoint
- `/Tasks` shows the full Fetch API CRUD demo
- `/Privacy` explains local-only data handling and the development CORS note
- `/api/studytasks` returns the task list as raw JSON

## Build Check

```bash
dotnet build
```
