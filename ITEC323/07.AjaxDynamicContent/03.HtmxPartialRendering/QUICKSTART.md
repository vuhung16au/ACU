# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 03.HtmxPartialRendering
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the HTMX learning goals
- `/HtmxBoard` shows the main fragment-rendering demo
- `/Privacy` explains the local offline teaching approach
- `/fragments/notices` returns one HTML fragment response used by the page

## Build Check

```bash
dotnet build
```
