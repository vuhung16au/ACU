# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 04.LocalTodoApi
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the local todo API learning goals
- `/TodoDashboard` shows the full local CRUD demo
- `/Privacy` explains local-only data handling and the development CORS note
- `/api/todoitems` returns the todo list as raw JSON

## Build Check

```bash
dotnet build
```
