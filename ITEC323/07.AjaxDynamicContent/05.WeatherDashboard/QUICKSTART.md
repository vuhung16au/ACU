# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 05.WeatherDashboard
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the mock weather dashboard learning goals
- `/Dashboard` shows the live mock weather dashboard
- `/Privacy` explains the local simulation approach
- `/api/weatherdashboard` returns the dashboard JSON payload

## Build Check

```bash
dotnet build
```
