# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 02.PartialPageUpdates
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the partial update learning goals
- `/Dashboard` shows the main partial page updates demo
- `/Privacy` explains the project scope and local-only data usage
- `/api/learningdashboard/announcements` returns one of the JSON feeds used by the page

## Build Check

```bash
dotnet build
```
