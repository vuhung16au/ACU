# Key Takeaways

- Use Fetch API when the browser needs to control JSON parsing and UI rendering.
- Use HTMX-style fragment requests when the server can return final HTML for a small targeted update.
- Use polling for lightweight dashboard information that needs regular refreshes.
- Configure CORS in development when APIs may be called from different local origins.
- Swagger UI makes local APIs easier to inspect, test, and explain to beginners.
