# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 07.ComprehensiveApp
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the capstone learning goals
- `/Dashboard` shows the integrated Fetch, polling, and HTMX-style demo
- `/Privacy` explains the local-only scope and development setup
- `/swagger` opens the API documentation in development
- `/api/capstonetasks` returns the capstone task list as raw JSON

## Build Check

```bash
dotnet build
```
