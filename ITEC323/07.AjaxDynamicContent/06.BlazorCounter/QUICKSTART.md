# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 06.BlazorCounter
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the Blazor counter learning goals
- `/counter` shows the shared counter component
- `/privacy` explains the local state approach

## Build Check

```bash
dotnet build
```
