# 3D Split Flap Departure Board

## Overview

This project shows a 3D split-flap style departure board with these lines:

- `ACU`
- `ITEC`
- `NEW UI`
- `DROPPED`

Watch a [Demo on YouTube](https://youtu.be/-uFvb6gBrs4)

## Run

Open [index.html](index.html) in a browser.

## Record With Playwright

The recording script opens the demo, plays the animation, and saves a `.webm` file in `artifacts/`.

Run:

```bash
npm install
npm run record
```
