# DotNetHelloWorldCLI

A simple .NET console application that demonstrates the basic structure of a C# program.

## Overview

This project is a basic "Hello, World!" console application written in C# using .NET. It serves as an introductory example for learning .NET console application development.

## What It Does

When executed, the application prints "Hello, World!" to the console. This demonstrates the fundamental structure of a .NET console application.
