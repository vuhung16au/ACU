# 34-React-SPA-Integration

## Overview

This module introduces Single Page Application integration by pairing a React frontend with an ASP.NET Core backend.

The module uses `Microsoft.JavaScript.Templates` with the `reactwebapi` template because it is the supported template-based React option available in this repository's `.NET 10` toolchain.

## Learning Objectives

By working through this module, students will learn how to:

- scaffold a React and ASP.NET Core sample with `reactwebapi`
- understand local development flow versus published deployment flow
- connect React components and state to ASP.NET Core API endpoints
- compare React and Angular approaches for ASP.NET Core integration
