# 01.reactwebapidemo.client

This React app is the frontend for the `01.ReactWebApiDemo` sample.

## Local Development

The ASP.NET Core backend uses SPA proxy support to launch the React development server when you run the server project in development mode.

## Main Frontend Files

- `src/App.tsx` - page state and user actions
- `vite.config.ts` - dev server configuration and API proxy
- `src/index.css` and `src/App.css` - lesson page styling

## Build

```bash
npm install
npm run build
```
