# 06.ComprehensiveMinimalApi

## Overview

This project combines the earlier lessons into one polished Tasks API.

## Learning Objectives

- combine routing, query strings, request bodies, validation, CRUD, and status codes
- use a simple repository and service structure
- build a small REST API that is still easy for beginners to follow

## Main Features

- `GET /api/tasks`
- `GET /api/tasks/{id}`
- `GET /api/tasks/search?priority=High&completed=false`
- `GET /api/tasks/summary`
- `POST /api/tasks`
- `PUT /api/tasks/{id}`
- `DELETE /api/tasks/{id}`
