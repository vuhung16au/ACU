# 22-MinimalAPIs-REST

## Overview

This folder is a planned showcase for building lightweight, high-performance REST APIs using **.NET 10 Minimal APIs**.

The focus is on helping you understand how to create simple HTTP APIs without the extra structure of MVC or Razor Pages.

## What You'll Learn

- what a REST API is
- how Minimal APIs map routes directly in `Program.cs`
- how to handle `GET`, `POST`, `PUT`, and `DELETE` requests
- how to read route values, query string values, and JSON request bodies
- how to return status codes and JSON responses
- how to validate simple input
- how to build small CRUD-style APIs

## Technology Stack

- .NET 10
- C# 14
- ASP.NET Core Minimal APIs
- JSON
- REST-style HTTP endpoints

## Projects

- `01.HelloMinimalApi`
- `02.RouteAndQueryParameters`
- `03.RequestBodiesAndValidation`
- `04.CrudWithInMemoryData`
- `05.ErrorHandlingAndStatusCodes`
- `06.ComprehensiveMinimalApi`

Each project builds on the previous one while keeping the examples short and approachable for first-time .NET students.

## Goal

The goal is to help you understand how to build small, fast, beginner-friendly web APIs with minimal ceremony, while still learning good REST fundamentals and clear endpoint design.
