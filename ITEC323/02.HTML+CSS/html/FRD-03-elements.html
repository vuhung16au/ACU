Attributes

Elements can have attributes that give extra information. The id attribute (e.g. id="attributes-section") uniquely identifies an element. The class attribute (e.g. class="intro") groups elements for styling.

Links use the href attribute: Example link. The title attribute shows a tooltip when you hover.
