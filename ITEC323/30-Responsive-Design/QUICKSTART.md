# QUICKSTART: 30-Responsive-Design

This folder is currently in planning.

## Planned Prerequisites
- .NET 10 SDK
- a code editor such as Visual Studio Code or Visual Studio
- a modern web browser

## Planned Build Flow
When the sample projects are implemented later, the usual flow will be:

```bash
cd 30-Responsive-Design
dotnet build
```

To run an individual sample later:

```bash
dotnet run --project 30-Responsive-Design/<ProjectFolder>
```

## Planned Testing Approach
- run the app locally
- open it in the browser
- resize the browser window
- check mobile, tablet, and desktop layouts

## Expected Outcome
Students should be able to compare how the same page adapts across different screen sizes using responsive design techniques.
