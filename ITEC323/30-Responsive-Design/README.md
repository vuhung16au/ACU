# 30-Responsive-Design

## Overview
This folder is a planned showcase for responsive web design using ASP.NET Core Razor Pages, HTML, and CSS.

The focus is on building web apps that work well across:
- mobile phones
- tablets
- laptops
- desktop screens

## What You'll Learn
- mobile-first design
- fluid layouts
- responsive spacing and typography
- Flexbox and CSS Grid
- responsive navigation
- responsive forms
- dashboard-style layouts

## Technology Stack
- .NET 10
- C# 14
- ASP.NET Core Razor Pages
- HTML5
- CSS3
- light JavaScript where needed

## Planned Structure
- `01.ResponsiveBasics`
- `02.FlexboxAndGrid`
- `03.ResponsiveNavigation`
- `04.ResponsiveForms`
- `05.ResponsiveDashboard`

## Goal
The goal is to help students understand how to design one web app experience that adapts cleanly to different screen sizes without building separate mobile and desktop apps.
