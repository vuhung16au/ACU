# 03.MaterialDesign3

A beginner Android app for practising Material Design 3 with Jetpack Compose.
## Demo 

![MaterialDesign3 demo](images/material-design-3.gif)

## What You Build
- A screen using `Scaffold`, `TopAppBar`, and `FloatingActionButton`
- Material 3 `Card` components for content
- A custom theme with light and dark modes

## Learning Focus
- Use Material 3 layout structure
- Apply a custom color scheme
- Toggle the UI between light and dark themes

## Project Files
- `app/src/main/java/` contains the Kotlin code
- `app/src/main/res/` contains strings and theme resources
- `docs/Key-Takeaways.md` summarises the main ideas

## Expected Result
When the app runs, the theme switch changes the screen style and the FAB adds a new update card.
