# 06.ComprehensiveApp

A beginner Android app that combines navigation, forms, and ViewModel state.

## Demo

![ComprehensiveApp demo](images/comprehensive-app.gif)

## What You Build
- A welcome screen and student list screen
- A form screen for adding and editing students
- A detail screen with edit and delete actions

## Learning Focus
- Combine navigation with a ViewModel
- Perform simple CRUD actions in memory
- Validate form input before saving

## Project Files
- `app/src/main/java/` contains the Kotlin code
- `app/src/main/res/` contains strings and theme resources
- `docs/Key-Takeaways.md` summarises the main ideas

## Expected Result
When the app runs, the user can add, edit, view, and delete students while the data stays available until the app closes.
