# Week 10: Android Development 1 - Kotlin & Jetpack Compose

## Overview
Introduction to Android development using Kotlin and Jetpack Compose for Android 16 (Baklava).

## Learning Objectives
- Set up Android Studio development environment
- Understand Android platform and app architecture
- Build Android apps with Kotlin programming language
- Create modern UIs with Jetpack Compose
- Implement Material Design 3 (Material You)
- Use ViewModel for state management
- Navigate between screens in multi-screen apps

## Why Android?
- **Market Leader**: 70%+ global mobile OS market share
- **Massive Reach**: Billions of smartphone users worldwide
- **Modern Stack**: Kotlin-first with Jetpack Compose
- **Career Ready**: High demand for Android developers

## Technology Stack
- **Language**: Kotlin
- **UI Toolkit**: Jetpack Compose (declarative UI)
- **Target Platform**: Android 16 (API Level 36)
- **Minimum SDK**: API 26 (Android 8.0)
- **IDE**: Android Studio with Gemini AI assistant
- **Architecture**: MVVM with Jetpack components

## Essential Concepts

### Android Platform
- Linux-based open-source mobile OS
- ART (Android Runtime) for app execution
- Rich framework APIs for device features
- Google Play Services integration

### Jetpack Compose
- **Declarative UI**: Describe what, not how
- **Less Code**: Simpler than XML layouts
- **Live Previews**: See changes instantly
- **Material Design 3**: Built-in modern components
- **Kotlin-first**: Designed for Kotlin from the ground up

### Key Jetpack Components
- **Compose UI**: Modern toolkit for building native UI
- **ViewModel**: Lifecycle-aware data management
- **Navigation**: Type-safe screen navigation
- **Room**: Local database (covered in Week 11)
- **WorkManager**: Background tasks (covered in Week 11)
