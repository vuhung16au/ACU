# Quick Start

## Run

```bash
cd 06.ComprehensiveExample
dotnet restore
dotnet build
dotnet run
```

Open:

- `https://localhost:5001`
- `http://localhost:5000`

## Explore

1. `/` home page with reusable hero and cards.
2. `/Projects` and `/Blog` pages reuse shared partials.
3. `/Admin` uses folder-level admin layout.
4. `/PrintProfile` uses page-level print layout override.
