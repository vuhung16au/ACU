# Quick Start

## Prerequisites

- .NET 10 SDK
- macOS, Windows, or Linux terminal

## Run

```bash
cd 04.PartialViews
dotnet restore
dotnet build
dotnet run
```

Open:

- `https://localhost:5001`
- `http://localhost:5000`

## What To Check

1. Home page renders alert, cards, and sidebar partials.
2. About page explains when partials are useful.
3. Contact page reuses the same alert and sidebar patterns.
4. Navigation and footer come from shared partials inside the layout.
