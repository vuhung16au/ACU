# Quickstart - 06.BlazorInteractive

## Prerequisites

- .NET SDK 10 preview (matching solution projects)

## Run

From the workspace root:

```bash
dotnet run --project 06.BlazorInteractive/06.BlazorInteractive.csproj
```

Open the app URL shown in the terminal and navigate to:

- `/`
- `/BlazorLab`

## What to try

1. Use the counter buttons.
2. Change score and weight in the grade calculator.
3. Complete the planner wizard steps.
4. Observe cascading cohort context.
5. Click "Load announcements" to fetch API data.
