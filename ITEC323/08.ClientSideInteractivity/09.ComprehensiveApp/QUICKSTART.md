# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run the project

```bash
cd 08.ClientSideInteractivity/09.ComprehensiveApp
dotnet run
```

Open the local URL shown in the terminal (for example, `https://localhost:5xxx`).

## Pages to check

- `/` &mdash; overview page that:
  - Summarises vanilla JS, Alpine.js, HTMX, Blazor, React, and Vue.
  - Explains when each approach is a good fit.
  - Suggests practice ideas for implementing the same feature in different ways.

## Build check

```bash
dotnet build
```

## Suggested classroom flow

1. Run this comparison app and read through the overview cards on the home page.
2. For each approach, briefly revisit the matching project (01&ndash;08) in this module.
3. As a group, choose one feature (for example, a small study planner or todo list).
4. Implement that feature with at least two different approaches (for example, HTMX and React).
5. Discuss the experience: which tool felt simplest, and which gave the most flexibility?
