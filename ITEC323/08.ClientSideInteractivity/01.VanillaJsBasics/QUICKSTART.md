# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 08.ClientSideInteractivity/01.VanillaJsBasics
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` module home page with objectives and vanilla JS tips
- `/Playground` hands-on DOM, event, classList, and form validation demo
- `/Privacy` notes about local-only demo data handling

## Build Check

```bash
dotnet build
```

## Classroom Flow Suggestion

1. Start on `/Playground` and select a demo card.
2. Toggle emphasis mode to observe class changes.
3. Type text to compare `textContent` and `innerHTML` behavior.
4. Submit the student form and inspect dynamic DOM updates.
