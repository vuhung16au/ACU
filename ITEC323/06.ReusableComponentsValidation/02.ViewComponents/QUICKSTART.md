# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 02.ViewComponents
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` shows all three components together
- `/Dashboard` passes a `userId` to the profile and recent items components
- `/Activity` reuses the same components with different parameter values

## Build Check

```bash
dotnet build
```
