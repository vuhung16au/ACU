# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 06.ComprehensiveApp
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` shows the dashboard and sidebar components
- `/Users` lists all CSV-backed users
- `/Create` adds a user
- `/Details/{id}` shows safe and sanitized bio output
- `/Edit/{id}` updates a user
- `/Delete/{id}` removes a user

## Build Check

```bash
dotnet build
```
