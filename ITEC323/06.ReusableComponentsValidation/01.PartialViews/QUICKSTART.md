# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 01.PartialViews
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` shows the shared header, footer, and featured product cards
- `/Products` loops through products with the same `_ProductCard` partial
- `/Contact` renders the reusable `_ContactForm` partial

## Build Check

```bash
dotnet build
```
