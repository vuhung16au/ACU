# Quick Start

## Prerequisites

- .NET 10 SDK installed

## Run The Project

```bash
cd 04.AdvancedValidation
dotnet run
```

Open the local URL shown in the terminal.

## Pages To Check

- `/` explains the advanced validation examples
- `/Register` shows regex, compare, and custom age validation
- `/Security` shows safe, unsafe, and sanitized HTML output

## Build Check

```bash
dotnet build
```
